# Astra Non-Goals

## Status

Precode foundation document.

## Purpose

This document defines what Astra v0.1 is not trying to do.

## Non-Goals

Astra v0.1 is not:

1. a fully autonomous software engineer;
2. a replacement for human product ownership;
3. a replacement for human code review;
4. a replacement for Scrum accountabilities;
5. a production deployment automation system;
6. an LLM provider SDK;
7. a project management clone;
8. a generic chatbot shell;
9. a cloud platform;
10. a substitute for tests, verification, or review.

## Design Guardrail

If a feature requires Astra to make consequential decisions without human approval, it is out of scope for v0.1.
